Steps Data Preprocessing key steps data preprocessing Data Cleaning , Data Integration , Data Transformation , Data Reduction . 1 . Data Cleaning : process identifying correcting errors inconsistencies dataset . involves handling missing values , removing duplicates , correcting incorrect outlier data ensure dataset accurate reliable . Clean data essential effective analysis , improves quality results enhances performance data models . Missing Values : occur data absent dataset . ignore rows missing data fill gaps manually , attribute mean , probable value . ensures dataset remains accurate complete analysis . Noisy Data : refers irrelevant incorrect data difficult machines interpret , caused errors data collection entry . handled ways : Binning Method : data sorted equal segments , segment smoothed replacing values mean boundary values . Regression : Data smoothed fitting regression function , linear multiple , predict values . Clustering : method groups similar data points , outliers undetected falling outside clusters . techniques help remove noise improve data quality . Removing Duplicates : involves identifying eliminating repeated data entries ensure accuracy consistency dataset . process prevents errors ensures reliable analysis keeping unique records . 2 . Data Integration : involves merging data sources single , unified dataset . challenging differences data formats , structures , meanings . Techniques like record linkage data fusion help combining data efficiently , ensuring consistency accuracy . Record Linkage process identifying matching records different datasets refer entity , represented differently . helps combining data sources finding corresponding records based common identifiers attributes . Data Fusion involves combining data multiple sources create comprehensive accurate dataset . integrates information inconsistent incomplete different sources , ensuring unified richer dataset analysis . 3 . Data Transformation : involves converting data format suitable analysis . Common techniques include normalization , scales data common range ; standardization , adjusts data zero mean unit variance ; discretization , converts continuous data discrete categories . techniques help prepare data accurate analysis . Data Normalization : process scaling data common range ensure consistency variables . Discretization : Converting continuous data discrete categories easier analysis . Data Aggregation : Combining multiple data points summary form , averages totals , simplify analysis . Concept Hierarchy Generation : Organizing data hierarchy concepts provide higher - level view better understanding analysis . 4 . Data Reduction : reduces dataset size maintaining key information . feature selection , chooses relevant features , feature extraction , transforms data lower - dimensional space preserving important details . uses reduction techniques , Dimensionality Reduction ( e.g. , Principal Component Analysis ) : technique reduces number variables dataset retaining essential information . Numerosity Reduction : Reducing number data points methods like sampling simplify dataset losing critical patterns . Data Compression : Reducing size data encoding compact form , making easier store process .
